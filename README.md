# OpportCI 

Landing page de génération de leads pour **OpportCI**, une newsletter 
100% ivoirienne destinée aux étudiants de Bac+1 à Bac+3 qui cherchent 
leur premier stage en Côte d'Ivoire.

Chaque semaine, les abonnés reçoivent :
- 1 offre de stage sélectionnée
- 1 conseil CV & entretien
- 1 parcours d'étudiant inspirant

## Fonctionnalités du site
- Formulaire de capture d'email (hero + section CTA)
- Animations au scroll sur toutes les sections
- Section témoignages et statistiques
- Design responsive mobile / desktop
- Navigation fluide entre les sections

## Technologies utilisées
- HTML5
- CSS3
- JavaScript (Vanilla) — animations Intersection Observer
- FontAwesome 6
- Google Fonts (Inter, Roboto)

## Ce que j'ai appris
- Conception d'une page orientée conversion (copywriting, CTA, 
  social proof)
- Implémentation d'animations au scroll avec l'API 
  Intersection Observer
- Structure d'une landing page de génération de leads
- Application des principes marketing (AIDA) dans la conception web
